# Sales Data Analyst Project (Power BI)

This project analyzes sales data using Power BI to track sales trends, region-wise and product-wise performance, and overall business insights. Interactive dashboards help in data-driven decisions.

## Tools Used
- Power BI Desktop
- Excel
- Git & GitHub

## Dashboard Features
- Bar Charts
- Pie Charts
- KPIs & Cards
- Filters & Slicers
